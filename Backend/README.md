# COMO INICIAR EL SERVER:

* Estar parado en la raiz de Backend y en la terminar correr este comando para instalar paquetes:

- npm install

* Dentro del archivo server.js cololar las credenciales correctas para la conexion a la base de datos
* Para correr el server:

- npm run start

* Dentro de la carpeta request se encuentran las consultas http
* es necesario installar la extension

Name: REST Client

Description: REST Client for Visual Studio Code
